<li><code>def runRepositoryTests(gitHubRepo, packagesList, dependentJobs, rootFolder, buildOptions, testTargetName='all_checks_and_tests', cleanup=true)</code>High Level testing for rez projects. <ul><b>gitHubRepo</b> full url to the github repository	<b>packagesList</b> the list of packages to process	<b>dependentJobs</b> The fullpath of a dependent multibranch job	<b>rootFolder</b> Where to install the build artifacts(rez packages).	<b>buildOptions</b> the flags to be passed to the rez build command.</ul></li>
<li><code>def runPackageTestsSequentially(sequentialPackages, options, installPath)</code> Runs the tests of a list of rez-packages (inside a git-repo) in sequence.<ul><b>sequentialPackages</b> a list of the relative (to the repository root) paths to a folder containing a package (ex. ['src/AL/libs/scenery','src/AL/libs/scenerymaya'] ).<b>buildOptions</b> the rez build options passed to the 'rez build -- tests' command . For example:  " -i -c -- -Dsymlink=OFF -- ". <b>installPath</b> Where to save the rez artifacts </ul></li>
<li><code>def runPackageTestsInParallel(parallelPackages, options, installPath)</code> Runs the tests of a list of rez-packages (inside a git-repo) in parallel.<ul><b>parallelPackages</b> a list of the relative (to the repository root) paths to a folder containing a package (ex. ['src/AL/libs/scenery','src/AL/libs/scenerymaya'] ).<b>buildOptions</b> the rez build options passed to the 'rez build -- tests' command . For example:  " -i -c -- -Dsymlink=OFF -- ". <b>installPath</b> Where to save the rez artifacts </ul></li>
<li><code>def runPackageTests(packages, options, installPath, mode, testTargetName)</code> Wraps runPackageTestsSequentially and runPackageTestsInParallel<ul><b>mode</b>either 'serial' or 'parallel'</ul></li>
